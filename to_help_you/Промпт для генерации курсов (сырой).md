## Контекст
- Курс имеет заголовок, краткое описание. Цели / навыки, которые он разваивает
- Курс состоит из уроков. Каждый урок состоит из карточек. Карточка либо для практики, либо для теории. 
- Карточка теории. Может иметь текст, картинку.
- Карточка практики. Задаёт один вопрос. Формат ответа: выбора из множества, выбор нескольких, да / нет, ответ текстом.
- Карточка строго привязана к уроку.
- Карточка практики также имеет подсказки от 1 до 5.
